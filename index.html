Sơ đồ phát triển năng lực SOC Analyst

Lộ trình này dùng để định hướng portfolio: mỗi giai đoạn đều có kiến thức cần học, lab cần làm và bằng chứng nên đưa lên GitHub Pages.

Blue Team Automation

Python scripts, enrichment, report automation, SOAR mindset.

Evidence: automation mini-projects

Short-term: 0–3 tháng

  • Hoàn thiện ảnh lab cho Wazuh, ELK/Snort và CCNA Packet Tracer.
  • Viết thêm 3 case SOC: malicious IP, brute force SSH, phishing email.
  • Chuẩn hóa mỗi bài theo format: Objective → Architecture → Evidence → Assessment → Recommendation.

Mid-term: 3–6 tháng

  • Thêm detection rule mapping MITRE ATT&CK cho từng use case.
  • Tạo dashboard theo severity, source IP, destination, rule name và timeline.
  • Thực hành viết báo cáo điều tra ngắn như SOC Analyst Tier 1.

Long-term: 6–12 tháng

  • Xây dựng mini SOAR/automation flow: enrich IOC, tạo ticket, gợi ý action.
  • Mở rộng sang threat hunting cơ bản và incident response playbook.
  • Chuẩn bị portfolio ứng tuyển SOC Analyst L1 / NOC / Network Security Intern.
