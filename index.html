SOC Workflow

Quy trình điều tra dùng xuyên suốt portfolio

Mỗi case trong portfolio được trình bày theo một workflow thống nhất để người đọc thấy rõ tư duy điều tra thay vì chỉ thấy ảnh giao diện tool.

01Intake

Ghi alert name, severity, affected asset, user, source, destination và timestamp.

02Validate

Kiểm tra raw log, rule match, time window và data source.

03Correlate

Pivot qua endpoint, network, DNS/proxy, WAF, email hoặc authentication log.

04Assess

Phân loại true positive, false positive, benign activity hoặc needs escalation.

05Respond

Đề xuất monitor, tune rule, block IOC, isolate host, reset password hoặc escalate.
