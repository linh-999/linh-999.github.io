Hướng dẫn thêm CV vào website

1. Xuất CV của bạn ra file PDF.
2. Đổi tên file thành: cv.pdf
3. Đặt file vào đúng thư mục này: documents/cv.pdf
4. Push lại project lên GitHub.

Khi bấm nút CV trên website, trình duyệt sẽ mở file documents/cv.pdf.
Lưu ý: GitHub Pages phân biệt chữ hoa/thường, nên tên file phải đúng là cv.pdf.
