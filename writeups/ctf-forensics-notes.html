🔎 CTF Forensics Study Notes: Quy trình phân tích bằng chứng

Ghi chú cách tiếp cận challenge forensics: xác định loại file, kiểm tra metadata, strings, timeline, hidden data và viết lại solve path.

CTFForensicsWrite-upEvidence

Mục tiêu

  • Có checklist giải bài Forensics CTF từ cơ bản đến trung bình.
  • Biết cách ghi lại bằng chứng thay vì chỉ ghi flag.
  • Rèn khả năng trình bày solve path rõ ràng.

Checklist phân tích

  • Xác định loại file bằng file command hoặc signature.
  • Kiểm tra metadata bằng exiftool nếu là ảnh/tài liệu.
  • Dùng strings để tìm chuỗi đáng chú ý.
  • Kiểm tra archive, hidden file, steganography hoặc dữ liệu bị nối cuối file.
  • Lập timeline nếu challenge liên quan log, disk image hoặc mobile backup.

Cách viết write-up

  • Nêu đề bài và mục tiêu.
  • Liệt kê công cụ sử dụng.
  • Ghi từng bước phân tích và ảnh/bằng chứng.
  • Giải thích vì sao chọn hướng tiếp cận đó.
  • Kết luận flag hoặc kết quả tìm được.

Công cụ thường dùng

  • file, strings, binwalk, exiftool.
  • Wireshark cho pcap.
  • Autopsy/FTK Imager cho disk image.
  • SQLite Browser cho database.
  • CyberChef cho decode/convert dữ liệu.

Lỗi thường gặp

  • Nhảy thẳng vào tool mà không xác định file type.
  • Không ghi lại command đã chạy nên khó viết lại write-up.
  • Chỉ ghi flag, thiếu giải thích quá trình tìm ra flag.

Kỹ năng rút ra

  • Tư duy điều tra theo bằng chứng.
  • Khả năng dùng công cụ forensics cơ bản.
  • Viết báo cáo kỹ thuật ngắn gọn và dễ đọc.

Command / Query tham khảo

file challenge.bin
strings challenge.bin | head
exiftool image.png
