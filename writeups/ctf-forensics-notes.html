🔎 CTF Forensics Study Notes: Quy trình phân tích bằng chứng

Checklist phân tích Forensics CTF theo hướng bằng chứng: file type, metadata, strings, embedded data, timeline và conclusion.

CTFForensicsEvidenceWrite-upTimeline

Study Note Summary

Ghi chú Forensics CTF theo hướng rèn tư duy bằng chứng. Thay vì chỉ ghi flag, write-up cần mô tả đường đi phân tích, công cụ dùng, evidence tìm được và lý do kết luận.

DateDec 2024
FocusForensics evidence
Toolsfile, strings, exiftool, binwalk
OutputChecklist + solve path

1. Checklist

  • Xác định loại file thật bằng magic bytes, không chỉ dựa vào extension.
  • Kiểm tra metadata, timestamp, author/tool nếu có.
  • Dùng strings để tìm text, URL, path hoặc flag fragment.
  • Kiểm tra embedded file bằng binwalk/foremost nếu nghi có file ẩn.
  • Tạo timeline nếu có nhiều log/file liên quan.
  • Ghi lại lệnh đã chạy và output quan trọng.

2. Report Format

SectionContent
Initial observationFile name, size, type, hint.
Analysis stepsLệnh/công cụ đã dùng và lý do dùng.
EvidenceChuỗi, metadata, hidden file, timestamp liên quan.
ConclusionFlag hoặc finding và cách xác minh.

3. SOC Relevance

Forensics CTF giúp luyện cách không bỏ sót bằng chứng và không kết luận khi chưa có dữ liệu. Kỹ năng này chuyển sang SOC rất tốt: khi điều tra alert, analyst cũng cần timeline, evidence, hypothesis và conclusion rõ ràng.
