# Phạm Huỳnh Khánh Linh Portfolio

Portfolio tĩnh chạy bằng GitHub Pages. Trang chủ được thiết kế ngắn gọn, các phần chi tiết được tách thành từng trang: Giới thiệu, Dự án, Bài lab, Ghi chú, Kỹ năng và CV. Phần liên hệ nằm trực tiếp ở trang chủ và footer.

## Upload nhanh

```bash
git status
git add .
git commit -m "Refine portfolio into multi-page layout"
git push origin main
```


## Cập nhật CV

Đặt file CV dạng PDF vào:

```text
documents/cv.pdf
```

Các nút CV trên website đã trỏ sẵn đến đường dẫn này.
