Executive Summary

Project tổng hợp các lab CCNA/Packet Tracer theo hướng phục vụ SOC/NOC: hiểu topology, xác định subnet, kiểm tra đường đi của traffic, đọc trạng thái interface và viết troubleshooting note. Đây là nền tảng quan trọng để SOC Analyst hiểu network log và phân biệt lỗi vận hành với dấu hiệu tấn công.

1. Lab Scope

  • VLSM addressing design for multiple LANs and WAN links.
  • Basic router/switch configuration: hostname, interface IP, no shutdown, VLAN 1 management.
  • Connectivity verification: ping, traceroute, routing table, interface status.
  • Troubleshooting: wrong mask, wrong default gateway, down interface, missing route.

2. SOC/NOC Value

Khi phân tích alert mạng, analyst cần biết một IP thuộc subnet nào, máy đó nằm ở VLAN nào, default gateway đúng không, traffic đi qua router/firewall nào. Vì vậy, lab CCNA không chỉ là bài cấu hình, mà là nền tảng để đọc network evidence trong SOC.
