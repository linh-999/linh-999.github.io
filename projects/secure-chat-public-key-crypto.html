01 · summary

Executive Summary

Project nghiên cứu RSA, Merkle-Hellman, ElGamal và xây dựng ứng dụng chat sử dụng mã hóa khóa công khai. Vai trò phù hợp để trình bày là nghiên cứu thuật toán, phát triển giao diện, kiểm thử và đánh giá bảo mật.

02 · concept

Security Concept

  • Khóa công khai dùng để mã hóa hoặc trao đổi thông tin an toàn.
  • Khóa riêng được giữ bởi người nhận để giải mã.
  • Mô hình giúp giải quyết bài toán chia sẻ khóa trong môi trường không tin cậy.
  • Ứng dụng chat là case dễ hiểu để minh họa confidentiality.
03 · flow

Application Flow

User login/registerGenerate/load key pairExchange public keyEncrypt messageSend via chat appDecrypt with private key

Trong portfolio, luồng này giúp người đọc thấy rõ mật mã được đặt vào ứng dụng như thế nào thay vì chỉ học công thức.

05 · limitations

Limitations

  • SimpleRSA phù hợp minh họa học thuật, không thay thế thư viện crypto production.
  • Cần bổ sung padding an toàn, xác thực khóa công khai và bảo vệ private key.
  • Cần TLS cho transport layer và hash mật khẩu đúng chuẩn.
  • Cần threat model cho MITM/key substitution.
06 · recommendations

Recommendations

  • Dùng thư viện crypto chuẩn nếu triển khai thực tế.
  • Bổ sung digital signature để xác thực người gửi.
  • Mã hóa private key khi lưu cục bộ.
  • Thiết kế cơ chế fingerprint/QR để xác minh khóa công khai.
07 · value

Portfolio Value

Learning value: Project này hỗ trợ nền tảng security fundamentals: confidentiality, key management, secure communication và secure design thinking.
