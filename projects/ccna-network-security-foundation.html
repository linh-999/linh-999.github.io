01 · Executive Summary

Executive Summary

Tổng hợp nền tảng mạng phục vụ SOC và Network Operations: subnetting, VLAN, routing, ACL, NAT, default gateway, troubleshooting bằng ping/traceroute/show commands. Project được trình bày theo hướng ứng dụng vào phân tích alert và xác định luồng traffic.

Security operations value: Nền tảng mạng giúp SOC Analyst hiểu source/destination, route path, segment, gateway và firewall/ACL impact khi điều tra sự cố.
02 · Scope

Objective & Scope

  • Nắm VLSM/subnet để đọc IP, mask và gateway trong alert.
  • Hiểu VLAN/routing để xác định segment bị ảnh hưởng.
  • Dùng ACL/NAT như biện pháp kiểm soát traffic.
  • Thực hành command verification để tìm lỗi kết nối.

05 · Triage process

Investigation Workflow

  1. Xác định IP/subnet của host trong alert.
  2. Kiểm tra gateway, VLAN và path đến destination.
  3. Xác minh ACL/NAT nếu traffic bị block hoặc đi sai hướng.
  4. Dùng ping/traceroute/show command để khoanh vùng lỗi.
  5. Ghi lại topology và kết luận network impact.

07 · Findings

Key Findings

  • Nhiều lỗi lab đến từ gateway/mask/VLAN không khớp.
  • SOC Analyst cần đọc được network path để hiểu alert không bị tách rời khỏi topology.
  • ACL/NAT vừa là công cụ bảo vệ vừa có thể gây false positive trong troubleshooting.
08 · Response

Recommendations

  • Lưu sơ đồ topology và addressing table cho mỗi lab.
  • Gắn network segment vào asset inventory.
  • Ghi rõ command evidence khi viết report network issue.
09 · Continuous improvement

Limitations & Next Improvements

  • Mở rộng sang OSPF, DHCP snooping và port security.
  • Tạo lab mô phỏng segmentation để hạn chế lateral movement.
  • Kết hợp Packet Tracer với Wireshark để quan sát packet flow.
Quality note: Kết luận trong report luôn bám vào dữ liệu hiện có; nếu thiếu log hoặc thiếu context, phần limitation được ghi rõ để giữ tính chính xác của case.
