Executive Summary

SOC Analyst muốn phân tích network alert tốt cần nền tảng mạng vững. Project này trình bày các lab CCNA theo hướng bảo mật/vận hành: thiết kế địa chỉ, cấu hình VLAN/routing/ACL, kiểm thử connectivity và troubleshooting theo OSI layer.

Objective & Scope

  • Thiết kế IP addressing/VLSM đúng yêu cầu số host.
  • Cấu hình router, switch, VLAN, inter-VLAN routing và ACL cơ bản.
  • Dùng command kiểm tra để chứng minh lab hoạt động.
  • Rèn tư duy troubleshooting layer 1–3 trước khi kết luận lỗi bảo mật.

Architecture / Log Flow

Alert / Event Source→Log Collection→SIEM / Dashboard→Analyst Triage→Verdict & Response
  • PC/Server kết nối switch access.
  • Switch chia VLAN và trunk về router/L3 switch.
  • Router xử lý routing, NAT hoặc ACL tùy lab.
  • Verification bằng ping, traceroute, show ip route, show vlan, show access-lists.
Chèn ảnh lab tại đây: topology, dashboard, rule, event detail hoặc command output.

Detection Logic / Use Cases

  • Không phải detection project; tập trung vào nền tảng để đọc network evidence.
  • ACL log có thể dùng như nguồn security event cơ bản.
  • Routing/VLAN sai có thể gây false symptom giống sự cố security.
# analyst-note
Detection should define: objective → data source → condition → expected evidence → false-positive notes → response action.

Investigation Workflow

  1. Đọc requirement và chia subnet bằng VLSM.
  2. Lập addressing table, gateway và mask cho từng LAN.
  3. Cấu hình interface/VLAN/routing/ACL.
  4. Kiểm tra bằng show command và end-to-end ping.
  5. Ghi lỗi gặp phải và cách sửa để tạo troubleshooting note.

Key Findings

  • Nền tảng subnet/routing giúp đọc log network chính xác hơn.
  • Sai gateway/mask/VLAN thường là nguyên nhân connectivity issue.
  • ACL cần test cả allow và deny để tránh cấu hình sai.

Recommendations

  • Ghi lại command verification trong mỗi lab report.
  • Tạo bảng lỗi thường gặp: wrong gateway, wrong mask, shutdown interface, VLAN mismatch.
  • Liên hệ từng lab với use case SOC như firewall deny, scan, segmentation.

Limitations & Next Improvements

  • Thêm Syslog/SNMP/NetFlow vào lab.
  • Tạo dashboard NOC/SOC cho thiết bị mạng.
  • Viết automation backup config bằng Python.

Limitation được ghi rõ để tránh phóng đại kết quả lab. Khi có ảnh/log thật, phần này nên được cập nhật bằng evidence cụ thể.
