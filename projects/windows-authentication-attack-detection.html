01 · Executive Summary

Executive Summary

Thiết kế logic phát hiện các hành vi tấn công xác thực trên Windows như brute force, password spray, successful login sau nhiều lần thất bại và quyền đặc biệt được cấp sau đăng nhập. Project tập trung vào cách đọc Event ID và correlation theo user, source IP, host và time window.

02 · Scope

Objective & Scope

  • Phân biệt brute force theo một user và password spray theo nhiều user.
  • Correlation Event ID 4625, 4624, 4672 và 4740.
  • Đánh giá rủi ro dựa trên account type, source IP và logon type.
  • Đề xuất lock account, reset password hoặc block source khi cần.

05 · Triage process

Investigation Workflow

  1. Xác định alert là brute force, password spray hay privilege login.
  2. Kiểm tra source IP nội bộ/ngoài, user affected và host affected.
  3. Pivot theo username để xem có successful login sau failed attempts không.
  4. Kiểm tra account criticality và thời gian hoạt động bất thường.
  5. Ghi verdict và đề xuất: monitor, reset password, disable account, block source hoặc escalate.

07 · Findings

Key Findings

  • Cần nhìn pattern theo time window, không đánh giá từng event riêng lẻ.
  • Password spray thường có count mỗi user thấp nhưng số user bị thử cao.
  • Successful login sau nhiều failed attempts là tín hiệu cần ưu tiên cao hơn.
08 · Response

Recommendations

  • Tách rule cho brute force và password spray để giảm false positive.
  • Bổ sung allowlist cho vulnerability scanner hoặc IAM test hợp lệ.
  • Cảnh báo riêng cho privileged account và service account.
09 · Continuous improvement

Limitations & Next Improvements

  • Thêm enrichment asset/user criticality từ CMDB.
  • Kết hợp VPN log để xác định source từ remote access.
  • Tạo dashboard failed/success login theo user, host và source IP.
