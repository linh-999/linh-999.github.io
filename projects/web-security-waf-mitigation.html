🧱 Web Security: Penetration Testing & WAF Mitigation

Lab kiểm thử OWASP Top 10, phân tích HTTP request/response và đề xuất rule WAF nhằm giảm rủi ro tấn công web.

FortiWeb WAFBurp SuiteOWASP ZAPDVWA

Tổng quan

Dự án tập trung vào kiểm thử bảo mật ứng dụng web ở mức lab, sau đó đối chiếu log web/WAF để hiểu cách payload bị phát hiện hoặc bị chặn.

Nội dung đã thực hiện

  • Dùng Burp Suite/OWASP ZAP để quan sát request và thử payload cơ bản.
  • Kiểm tra một số nhóm lỗi theo OWASP Top 10 như Broken Access Control, Injection và Security Misconfiguration.
  • Phân tích log để xác định endpoint, parameter, status code và rule bị match.
  • Đề xuất hướng tuning rule để giảm false positive.

Kết quả

  • Nắm được quy trình kiểm thử web cơ bản.
  • Hiểu cách liên hệ giữa tấn công web và log phòng thủ.
  • Biết trình bày vulnerability theo cấu trúc: mô tả, bằng chứng, impact và khuyến nghị.

Ghi chú: Nội dung này được viết theo hướng portfolio sinh viên/thực tập, có thể bổ sung ảnh lab, topology, screenshot dashboard hoặc log evidence sau khi có dữ liệu thực tế.
