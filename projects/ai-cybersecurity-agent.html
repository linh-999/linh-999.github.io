🤖 AI-Driven Cybersecurity Agent for Network Automation

Ý tưởng agent hỗ trợ phân tích packet/log và gợi ý bước điều tra cho SOC/NOC bằng Python, Scapy và LLM workflow.

PythonScapyLangChainWireshark

Tổng quan

Dự án nghiên cứu cách dùng AI assistant để hỗ trợ phân tích dữ liệu mạng và tự động hóa một phần quy trình điều tra. Mục tiêu không phải thay thế analyst, mà giúp tóm tắt dữ liệu và gợi ý bước kiểm tra tiếp theo.

Chức năng đề xuất

  • Đọc mô tả packet/log và tóm tắt dấu hiệu bất thường.
  • Gợi ý checklist điều tra dựa trên loại alert.
  • Sinh câu truy vấn hoặc command kiểm tra liên quan.
  • Tạo báo cáo ngắn gồm evidence, assessment và recommendation.

Kỹ thuật sử dụng

  • Python để xử lý dữ liệu.
  • Scapy/Wireshark để phân tích packet ở mức cơ bản.
  • LLM workflow để tóm tắt và hỗ trợ viết báo cáo.
  • Prompt template để chuẩn hóa đầu ra.

Kết quả mong muốn

  • Tăng tốc bước ghi chú và tổng hợp báo cáo.
  • Hỗ trợ sinh checklist cho SOC/NOC intern.
  • Rèn tư duy kết hợp network security, automation và AI.

Ghi chú: Nội dung này được viết theo hướng portfolio sinh viên/thực tập, có thể bổ sung ảnh lab, topology, screenshot dashboard hoặc log evidence sau khi có dữ liệu thực tế.
