# Project selection notes

Đã chọn các đồ án phù hợp nhất để đưa vào portfolio:

1. Multi-Agent Web Pentesting & Reporting — rất phù hợp với Security Automation / Pentest Intern.
2. Web Application Security Assessment — rất phù hợp với Web/AppSec, OWASP Top 10, Burp/ZAP.
3. Centralized IDS Monitoring with Snort & ELK — phù hợp SOC/SIEM.
4. FortiWeb WAF Defense Lab — phù hợp Web Defense / SOC alert triage.
5. Enterprise Network Design — phù hợp Network Security / NOC.
6. Remote Administration & Site-to-Site VPN Lab — phù hợp network security operations.
7. Secure Chat with Public-Key Cryptography — phù hợp nền tảng mật mã.
8. Wireless HID Security Awareness Lab — đưa vào theo hướng phòng thủ, không mô tả payload tấn công.
9. Voice Controlled Robot using STM32 & Bluetooth — đưa vào nhóm engineering/IoT, không đặt làm security project chủ lực.

ATTT-VAN.pdf là tài liệu học nền tảng, không đưa thành project riêng.
