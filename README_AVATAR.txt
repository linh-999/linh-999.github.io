CÁCH THAY AVATAR

1. Đặt ảnh của bạn vào thư mục assets/
2. Đặt tên ảnh là avatar.jpg hoặc avatar.png
3. Mở index.html và tìm dòng:
   <img src="assets/avatar-placeholder.svg" alt="Khung avatar Khánh Linh"/>
4. Đổi thành:
   <img src="assets/avatar.jpg" alt="Avatar Khánh Linh"/>

Lưu ý: GitHub Pages phân biệt chữ hoa/chữ thường, nên avatar.jpg khác Avatar.jpg.
